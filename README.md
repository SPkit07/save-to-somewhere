# 📊 Excel File Processor - Web Application

ระบบประมวลผลไฟล์ Excel พร้อมหน้าเว็บสำหรับตั้งค่า Path และลากปล่อยไฟล์

---

## 🎯 ลักษณะการทำงาน

### ด้านหน้าเว็บ (Frontend)
- **หน้าเดียวรวมทุกฟังก์ชัน**: แบ่งเป็น 2 ฝั่ง (ซ้าย-ขวา)
- **ฝั่งซ้าย**: 
  - แสดงชื่อสาขาอัตโนมัติเมื่อตรวจพบไฟล์
  - กล่อง Dropzone ขนาดใหญ่สำหรับลากปล่อยไฟล์ Excel
  - แสดงสถานะการประมวลผล
- **ฝั่งขวา**: 
  - ช่อง Input สำหรับกรอก Path ปลายทางแต่ละสาขา
  - บันทึกค่า Path ใน LocalStorage ของเบราว์เซอร์อัตโนมัติ

### ด้านเซิร์ฟเวอร์ (Backend)
- API ที่สร้างด้วย **FastAPI** เพื่อรับไฟล์ Excel และการตั้งค่า Path
- ประมวลผลไฟล์โดยใช้ Path ที่ผู้ใช้กรอก (override ค่าเดิม)
- บันทึกไฟล์ .txt ไปยังโฟลเดอร์ปลายทางตามที่ตั้งค่าไว้

## 🛠️ การติดตั้ง

### 1️⃣ ติดตั้ง Backend

```bash
# เปิด PowerShell หรือ Command Prompt
cd c:\Users\USER\Desktop\Save\ to\ somewhere

# สร้าง Virtual Environment
python -m venv venv

# เปิดใช้งาน Virtual Environment
venv\Scripts\activate

# ติดตั้ง Dependencies
pip install -r requirements.txt
```

### 2️⃣ เปิดใช้งาน Backend

```bash
# ยังอยู่ใน PowerShell ที่เปิด venv แล้ว
python app.py
```

Output ที่ควรเห็น:
```
INFO:     Uvicorn running on http://0.0.0.0:8000
```

✅ Backend ทำงาน! ให้ปล่อยหน้าต่างนี้เปิดไว้

### 3️⃣ เปิดหน้าเว็บ

- เปิด `index.html` ในเบราว์เซอร์ (Chrome, Firefox, Edge ฯลฯ)
- สามารถเปิดได้โดยการ:
  1. Double-click ไฟล์ `index.html` ใน Explorer
  2. หรือ: ใช้ VS Code Live Server extension

## 📝 วิธีใช้งาน

### ขั้นตอนการทำงาน

1. **ตั้งค่า Path (ครั้งแรกเท่านั้น)**
   - ขวาของหน้าจอ: กรอก Path สำหรับแต่ละสาขา
   - คลิก "💾 บันทึกการตั้งค่า" เพื่อบันทึก
   - ระบบจะจำค่านี้โดยอัตโนมัติในครั้งต่อไป

2. **ลากไฟล์ Excel**
   - ลากไฟล์ `.xlsx` มาปล่อยในกล่อง Dropzone ฝั่งซ้าย
   - หรือ คลิก "เลือกไฟล์" เพื่อเลือกจาก Explorer

3. **ระบบจะทำการ**
   - ตรวจพบสาขาโดยอัตโนมัติ
   - แสดงชื่อสาขาและรหัส
   - ประมวลผลและบันทึกไฟล์ .txt
   - แสดงผลลัพธ์ (จำนวนแถว ชื่อไฟล์ ฯลฯ)

## 🎨 ฟีเจอร์หลัก

### ✅ LocalStorage (จำข้อมูล)
- ระบบจะบันทึก Path ที่ผู้ใช้กรอกไว้ใน Browser Storage
- ครั้งต่อไปที่เปิดหน้าเว็บ ค่าเดิมจะแสดงอัตโนมัติ
- ไม่จำเป็นต้องกรอกซ้ำ

### ✅ Drag & Drop
- ลากไฟล์มาโดยไม่ต้องใช้ปุ่มเลือกไฟล์
- ตอบสนองฟีดแบ็ก (เปลี่ยนสี พื้นที่ Dropzone)

### ✅ การตรวจพบสาขา
- วิเคราะห์ Excel แล้วตรวจพบสาขาโดยอัตโนมัติ
- แสดงชื่อสาขา (K1, K2, K3, K4, K5, SP)

### ✅ การบันทึกไฟล์
- แยกบันทึกตามประเภท: หน้าร้าน (SP) และ โกดัง (WH)
- ใช้ Path ที่ผู้ใช้กรอก (override hardcode เดิม)
- ตั้งชื่อไฟล์อัตโนมัติพร้อมวันที่

## ⚙️ การปรับแต่ง

### เปลี่ยน URL ของ Backend

หากรัน Backend บนเซิร์ฟเวอร์อื่น แก้ไข `script.js`:

```javascript
// บรรทัด 3
const API_BASE_URL = 'http://192.168.1.100:8000'; // เปลี่ยนเป็น IP/URL ของเซิร์ฟเวอร์
```

### เปลี่ยน Port (พอร์ต)

หากต้องการรัน Backend บนพอร์ตอื่น แก้ไข `app.py`:

```python
# บรรทัดสุดท้าย
if __name__ == "__main__":
    import uvicorn
    uvicorn.run(app, host="0.0.0.0", port=9000, reload=True)  # เปลี่ยน 8000 เป็น 9000
```

และอัปเดตใน `script.js` ด้วย

## 🐛 แก้ปัญหา

### ❌ "Cannot reach API"
- ตรวจสอบว่า Backend ทำงานอยู่ (Terminal แสดง "running on http://0.0.0.0:8000")
- ตรวจสอบ URL ใน `script.js` ตรงกับ Backend หรือไม่

### ❌ "ไฟล์ Excel ไม่ถูกต้อง"
- ตรวจสอบว่าไฟล์มีคอลัมน์ "ReBplus" หรือไม่
- ตรวจสอบว่ามีค่า "0" ในคอลัมน์ "1=SP,2=WH" หรือไม่

### ❌ "Path ไม่ถูกต้อง"
- ตรวจสอบว่ากรอก Path แบบเต็มเส้นทาง (เช่น: `C:\Users\USER\Desktop\K1`)
- ตรวจสอบว่าโฟลเดอร์มีอยู่จริงหรือระบบมีสิทธิ์สร้างโฟลเดอร์
