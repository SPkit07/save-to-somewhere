# Column J & K Validation Feature - User Guide

## ⚡ New Feature

โปรแกรมตอนนี้มีฟีเจอร์ใหม่ในการตรวจสอบคอลัมน์ J และ K โดยอัตโนมัติ

## 🎯 What It Does

**ตรวจสอบว่าคอลัมน์ J (SP) และคอลัมน์ K (WH) ตรงกันหรือไม่**

- ✅ ตรวจสอบอัตโนมัติเมื่อประมวลผลไฟล์
- ✅ แสดงรายการสินค้าที่ไม่ตรงกันในรูปแบบตาราง
- ✅ แจ้งรหัสสินค้า (GOODS_CODE) และชื่อสินค้า (SKU_NAME)
- ✅ แสดงคำเตือนก่อนผลสรุป
- ✅ ยังคงประมวลผลแฟ้มต่อไปแม้มีปัญหา

## 📊 Table Example

```
⚠️ ตรวจพบความไม่ตรงกันในคอลัมน์ J และ K (3 รายการ)

+---+------------------+------------------+--------+--------+
| # | รหัสสินค้า       | ชื่อสินค้า       | Col J  | Col K  |
+---+------------------+------------------+--------+--------+
| 1 | SKU-2024-001     | สินค้า A         | 1      | 2      |
| 2 | SKU-2024-002     | สินค้า B         | 2      | 1      |
| 3 | SKU-2024-003     | สินค้า C         | 1      | 2      |
+---+------------------+------------------+--------+--------+
```

## 🔍 How to Read the Warning Table

| Column | ความหมาย |
|--------|---------|
| # | ลำดับที่ของปัญหา |
| รหัสสินค้า (GOODS_CODE) | รหัสสินค้าที่มีปัญหา |
| ชื่อสินค้า (SKU_NAME) | ชื่อสินค้าที่มีปัญหา |
| Col J (SP) | ค่าจากคอลัมน์ J |
| Col K (WH) | ค่าจากคอลัมน์ K |

**สีแดง** = ค่าไม่ตรงกัน

## 💡 Common Issues & Solutions

### ❌ Problem: "ตรวจพบความไม่ตรงกันในคอลัมน์ J และ K"

**สาเหตุ:**
- คอลัมน์ J (SP) มีค่า 1 แต่คอลัมน์ K (WH) มีค่า 2 (หรือตรงข้าม)
- ค่า null หรือค่าว่างในหนึ่งในคอลัมน์

**วิธีแก้:**
1. เปิดไฟล์ Excel
2. ค้นหารหัสสินค้าที่แสดงในตารางจากโปรแกรม
3. ตรวจสอบว่าคอลัมน์ J และ K มีค่าเดียวกันหรือไม่
4. แก้ไขให้ตรงกัน (ทั้งคู่ต้องเป็น 1 หรือ 2)
5. บันทึกไฟล์และประมวลผลใหม่

### ❌ Problem: "No Matching Products Found"

**วิธีแก้:**
- ตรวจสอบว่าไฟล์มีคอลัมน์ชื่อ GOODS_CODE หรือ SKU_NAME หรือไม่
- หากไม่มี โปรแกรมจะใช้คอลัมน์ F แทน

## ✅ What Works Well

- ✅ ตรวจสอบไฟล์ที่มี 1000+ สินค้า
- ✅ รองรับตัวเลข 1, 1.0, "1" (ทุกรูปแบบ)
- ✅ ลบช่องว่างโดยอัตโนมัติ
- ✅ แสดงทั้งคลัมน์ J และ K
- ✅ สนับสนุนหลายรายการที่มีปัญหา

## 🛠️ Configuration

ไม่ต้องตั้งค่าเพิ่มเติม - ทำงานอัตโนมัติ!

## 📋 Processing Flow

```
1. Upload Excel file
   ↓
2. System validates file
   ↓
3. Check Column J & K match
   ↓
4. If NOT matched → Show Warning Table ⚠️
   ↓
5. Continue processing (save files)
   ↓
6. Show final summary ✅
```
