🚀 เริ่มต้นใช้งาน (3 ขั้นตอน):

ขั้นตอนที่ 1: ติดตั้ง Dependencies (ครั้งแรกเท่านั้น)
─────────────────────────────────────────────────────

เปิด PowerShell แล้วรัน:

  cd "c:\Users\USER\Desktop\Save to somewhere"
  python -m venv venv
  venv\Scripts\activate
  pip install -r requirements.txt

ใช้เวลา ~2 นาที


ขั้นตอนที่ 2: เปิด Backend
─────────────────────────────

ทำได้ 2 วิธี:

วิธี A (ง่ายที่สุด):
  - Double-click: run-backend.bat

วิธี B (ด้วย PowerShell):
  - cd "c:\Users\USER\Desktop\Save to somewhere"
  - venv\Scripts\activate
  - python app.py

ตรวจสอบให้แน่ใจว่า Terminal แสดง:
  "Uvicorn running on http://0.0.0.0:8000"


ขั้นตอนที่ 3: เปิดหน้าเว็บและใช้งาน
──────────────────────────────────

1. Double-click: index.html
   (หน้าเว็บจะเปิดในเบราว์เซอร์)

2. ครั้งแรก: ตั้งค่า Path
   - กรอก Path สำหรับแต่ละสาขา (ฝั่งขวา)
   - คลิก "💾 บันทึกการตั้งค่า"

3. ครั้งหลังๆ: ลากไฟล์
   - ลากไฟล์ Excel → Dropzone (ฝั่งซ้าย)
   - รอประมวลผล
   - ✅ เรียบร้อย!

💡 Key Features:
═══════════════════════════════════════════════════════════════

🎯 Drag & Drop
   ลาก Excel file → Dropzone (ไม่ต้องเลือกจาก menu)

📦 LocalStorage
   ระบบจำ Path ที่ผู้ใช้กรอก ไม่ต้องกรอกซ้ำ

🏪 Auto Branch Detection
   ตรวจพบสาขา (K1, K2, K3, K4, K5, SP) อัตโนมัติ

📁 Path Override
   ใช้ Path ที่ผู้ใช้กรอก แทนค่า hardcode เดิม

✨ Auto Naming
   ชื่อไฟล์เสร็จพร้อมวันที่เรียบร้อย

🔐 Safe Processing
   ตรวจสอบข้อมูล ล้างข้อมูลไม่ถูก ป้องกัน error

🆘 Common Issues & Quick Fixes:
═══════════════════════════════════════════════════════════════

❌ "Cannot connect to API"
   → ตรวจสอบ Backend ทำงานหรือไม่

❌ "ไฟล์ไม่ถูกต้อง"
   → ตรวจสอบ Excel มีคอลัมน์ "ReBplus" หรือไม่

❌ "Path ไม่ถูกต้อง"
   → กรอก path แบบเต็ม เช่น C:\Users\USER\Desktop\K1

เพิ่มเติม: อ่าน TROUBLESHOOTING.md
